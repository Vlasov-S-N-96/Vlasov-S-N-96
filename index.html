Обо мне

Привет, я Сергей — работаю системным администратором в Воронежской областной клинической офтальмологической больнице (ВОКОБ). Сейчас активно учусь и развиваюсь в направлении Quality Assurance.


Из опыта работы: занимался разработкой и поддержкой базы данных (MySQL) для внутренних нужд больницы, писал SQL-запросы для отчётов, автоматизировал небольшие задачи на Python (обработка данных, перенос информации между системами). Знаком с основами тест-дизайна, составлением чек-листов и тест-кейсов, работой в Jira и Postman.


Ключевые достижения: разработал и поддерживал реляционную БД (MySQL) для больницы, автоматизировал перенос данных (ETL) между системами с помощью Python, создавал SQL-отчёты для руководства. Совмещаю опыт администрирования с практиками тест-дизайна, изученными на курсах.


Цель: найти позицию Junior QA Engineer, где смогу применять знания SQL, Python и основы тестирования. Готов много учиться, выполнять задачи от ручного тестирования до написания простых автотестов, помогая команде выпускать качественный продукт.
